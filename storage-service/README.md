# Storage service

## Build and Run

1. clone `qd-secrets` next to `qd-platform`:
```
- 2123
  |
  |- qd-secrets
  |
  |- qd-platform
    |- storage-service
      |- docker-compose.yml
      |- README.md
      |- ..
```

2. Navigate to the `storage-service` directory:
```
cd ./storage-service
```

3. To build run:
```
make build
```

4. To run the service:
```
make run
```
